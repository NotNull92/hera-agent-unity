# hera-agent-unity

Install the low-token CLI that lets AI coding agents control and verify a live Unity Editor:

```bash
npm install --global hera-agent-unity
hera-agent-unity doctor --json
```

The npm package installs the matching native CLI from the project's GitHub release. Add the Unity connector through Package Manager using:

```text
https://github.com/NotNull92/hera-agent-unity.git?path=AgentConnector
```

Documentation and source: <https://github.com/NotNull92/hera-agent-unity>
