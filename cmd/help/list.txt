Usage: hera-agent-unity list [options]

List registered tools (built-in + custom).

Default output is slim — one entry per tool with name and description only.
For full per-tool detail (output_schema + metadata) use --tool <name>.

Options:
  --names            Names only — flat JSON array (most token-efficient)
  --compact          Alias of --names for catalogue discovery
  --tool <name>      Full schema for a single tool

Examples:
  hera-agent-unity list
  hera-agent-unity list --names
  hera-agent-unity list --compact
  hera-agent-unity list --tool exec
