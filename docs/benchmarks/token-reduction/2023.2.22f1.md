# Token Reduction Benchmark — Unity 2023.2.22f1

> One-line summary: on Unity 2023.2.22f1, the v0.0.43 low-token discovery behavior matches the 2022.3 result: `list --compact` is 93 estimated tokens and `find_gameobjects --ids` is 54 estimated tokens in the realistic smoke scene.

- **Measured on**: 2026-06-22
- **CLI**: `hera-agent-unity v0.0.29`
- **Connector under test**: `AgentConnector/package.json` `0.0.43` as a local UPM package
- **Unity**: `2023.2.22f1`
- **Project**: `C:/Users/PC/Desktop/Cowork/Test2023.2.22f1`
- **Scene**: `Assets/HeraSmoke/Scenes/Smoke_2023_2.unity`
- **Token estimate**: UTF-8 output bytes divided by 4, rounded up

This benchmark measures the CLI output payload that comes back into an agent context. It does not include the outer tool-call framing overhead of a specific AI client.

## 5. Conclusion

On Unity 2023.2.22f1, the v0.0.43 token-reduction behavior matches the 2022.3 result:

- `list --compact` is 93 estimated tokens and is safe for bootstrap discovery.
- default `list` is still about 1857 estimated tokens and should be reserved for descriptive browsing.
- `find_gameobjects --ids` is 54 estimated tokens for a 20-result pickup search.
- `--fields all` remains available but costs 714 estimated tokens for the same 20-result search.

The low-token CLI path is stable across 2022.3.62f2 and 2023.2.22f1 for the tested smoke workflow.
